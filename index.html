Añadir vía repositorio

  1. En Kodi: Ajustes → Explorador de archivos → Añadir fuente.
  2. En <Ninguno> introducir esta dirección exacta:

    https://fullstackcurso.github.io/espadaily/
    Debe terminar en .io/espadaily/. La URL del navegador (github.com) devuelve la carpeta vacía.
  3. Asignar el nombre espadaily y aceptar.
  4. Ir a Add-ons → caja abierta (arriba a la izquierda) → Instalar desde archivo zip.
    (Si salta el aviso de seguridad: Ajustes → activar "Orígenes desconocidos").
  5. Seleccionar la fuente espadaily → repository.espadaily-1.0.2.zip.
  6. Cuando aparezca "Add-on instalado", ir a Instalar desde repositorio → EspaDaily Repository → Add-ons de vídeo → EspaDaily → Instalar.

Vía ZIP

  1. Descargar el .zip desde github.com/fullstackcurso/espadaily/releases
  2. En Kodi: Add-ons → caja → Instalar desde archivo zip → seleccionar el archivo descargado.
